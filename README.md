# feel2

## 專案說明
---
- 主程式: integration.py
- unit_test資料夾: 個別模組的單元測試py檔
- sounds開頭的資料夾: 音效檔(待整理)
- csv檔: 儲存色票, 播放次數, 對應音效, Code的對應資料表
- uniility資料夾: 存放檔案命名小工具

## 開發環境要求
---

- Python3.7
- Pydub
- Raspberry Pi Model 4B(Rasbian OS)
